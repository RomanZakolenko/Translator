### Simple Translator Pascal in assembly language.
 - Data Types: integer, real, boolean.  
 - Arithmetic operators: «-», «+», «*», «/», «shr», «shl».  
 - Relational operators: «=», «<>», «<», «>», «<=», «>=».  
 - Assignment operator: «:=».  
 - Structured statements: var, begin/end, if(else), for, writeln, sqrt.
